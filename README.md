# To-Do List API

API REST para gerenciamento de tarefas desenvolvida com Spring Boot.

## Tecnologias utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Funcionalidades

- Criar tarefa
- Listar tarefas
- Buscar tarefa por ID
- Atualizar tarefa
- Concluir tarefa
- Deletar tarefa
- Filtro de tarefas concluídas e pendentes

## Endpoints principais

POST /tasks  
GET /tasks  
GET /tasks/{id}  
PUT /tasks/{id}  
DELETE /tasks/{id}  
PUT /tasks/{id}/concluir  

## Banco de dados

Banco utilizado: MySQL
